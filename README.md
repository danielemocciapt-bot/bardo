# Bardo

PWA gratuita per colonne sonore di giochi di ruolo.

## Sviluppo
- `npm run dev` — server di sviluppo
- `npm test` — test unit (Vitest)
- `npm run build` — build di produzione in `dist/`

## Deploy (Cloudflare Pages, gratis)
- Comando di build: `npm run build`
- Cartella di output: `dist`
- Banda illimitata nel piano gratuito (adatto a servire audio).

## Contenuti audio
Gli audio in `public/audio/` sono royalty-free (Pixabay Content License); vedi `src/lib/data/licenses.json` per fonti e licenze. I contenuti eventualmente CC-BY richiedono attribuzione (schermata Credits prevista in una fase successiva).
